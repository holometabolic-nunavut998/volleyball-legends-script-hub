You need a computer running Windows 10 or Windows 11. Your computer must have a modern web browser installed. We recommend Google Chrome, Microsoft Edge, or Mozilla Firefox. You also need a browser extension to manage the script.

## 📥 How to Install the Script Manager

You need a userscript manager to run this tool. This extension acts as a bridge between your browser and the script.

1. Open your web browser.
2. Visit the Chrome Web Store or the Firefox Add-ons store.
3. Search for a tool named Tampermonkey.
4. Click the Add to Browser button.
5. Confirm the installation when prompted.
6. Look for the Tampermonkey icon in your browser toolbar to verify it works.

## 🛠 Troubleshooting Common Issues

If the script does not work, check these common fixes.

* Browser Compatibility: Ensure you use an updated version of your browser. Older versions often block script managers.
* Permissions: Click the Tampermonkey icon while you play. Ensure the toggle switch reads On. If it reads Off, click it once to restart the script.
* Page Reloads: Some game updates may cause the script to hang. Press F5 on your keyboard to reload the page. This forces the script to re-scan the game code.
* Conflicts: Other game extensions can interfere with the helper. Disable other game-related add-ons to see if the issue resolves.
